# Clever assessment

## Probably a better name: VV2 (Vurdering versjon 2)

A clickable experiment on what an assessment app in Osloskolen could look like

## Up and running

- git clone this repo
- cd into folder
- npm install
- Ensure that you have the correct .env file (including supabase url + key)
- npm run dev
